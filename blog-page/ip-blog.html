Whats an ip-address?


TL;DR

An ip-address is an address of a network. You can address things on a network with it.


IPv4 vs IPv6

Firs we have to differentiate between IPv4 and IPv6. Lucky for us to understand what an ip address is we don't have to differentiate a lot between them. What you should know though is some trivia about IPv4 and IPv6. IPv4 was the first kind of ip address we used, but soon, with the boom of the internet and the dotcom bubble, there were too many computers. So we ran out of IP addresses, this sounds dramatic but was acctually already known, so scientist prepared for this day and invented the IPv6 address. IPv6 addresses are made of hexadecimal numbers. With this we can give a trillion ip addresses to every square centimeter of the world, so thats devinetly enough for now.


What can you do with an IP address?

With an IP address you can address other computers or servers or pretty much anything in a network. Lets for example take the biggest network in the world, the internet. When you go to a website like for example github.com, then your computer looks up the ip address of the server the website github.com is stored on, your computer does that by going to a dns server and this server looks up the ip address for github.com or any domain. The DNS server then tells your computer the ip address of the domain that you searched for, in this case github.com, and your computer then addresses the server the website is stored on. The server the website is stored on, in this case the github servers, directs you to the starting page of the website.
